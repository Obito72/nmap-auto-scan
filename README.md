# 🔍 Nmap Auto Scan

A simple Bash script to automate common Nmap scans on a target system.

## 🛠️ What it does:
- ✅ Quick scan (`-T4 -F`)
- ✅ Service/version detection (`-sV`)
- ✅ Full TCP port scan (`-p-`)

## 📦 Requirements
- Linux (tested on Kali Linux)
- Nmap installed

## 🚀 Usage
```bash
chmod +x auto-nmap.sh
./auto-nmap.sh
